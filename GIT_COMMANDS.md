# Git Commands Reference

## Quick Push Command
```bash
git add . && git commit -m "message" && git push
```

## Usage Examples

### Basic commit and push
```bash
git add . && git commit -m "Update pipeline funnel" && git push
```

### Using the script
```bash
./git-push.sh "Your commit message here"
```

## Step-by-Step Commands

1. **Stage all changes:**
   ```bash
   git add .
   ```

2. **Commit with message:**
   ```bash
   git commit -m "Your commit message"
   ```

3. **Push to GitHub:**
   ```bash
   git push
   ```

## One-liner (all in one)
```bash
git add . && git commit -m "message" && git push
```
